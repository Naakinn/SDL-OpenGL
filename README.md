## Controls: 
- j/k - move cube backwards/towards
## Install dependencies 
- GLM: https://github.com/g-truc/glm?tab=readme-ov-file#build-and-install
## Build
```
$ cmake -B build
$ cmake --build build 
```
## Build in debug mode
```
$ cmake -B build -DDEBUG 
$ cmake --build build 
```
## Run 
```
$ ./engine
```
